# Security policy

This is a decommissioned educational lab, not a hosted service. Please do not open a public issue containing credentials, webhook URLs, account identifiers, private logs, or exploitable deployment details.

For a suspected secret exposure, revoke or rotate the value first, remove it from the current branch, and assess whether Git history must be rewritten. A closed or dismantled lab does not make historic identifiers appropriate portfolio evidence.

The reference implementation assumes disposable lab roles. Do not deploy it against production identities without independent threat modeling, rollback, session-revocation, monitoring, reconciliation, and change-control work.
